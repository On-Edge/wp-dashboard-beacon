Easily integrate a Helpscout Beacon in your dashboard or website.

== Description ==

This plugin makes it fast and easy to integrate Helpscout's Beacon feature in your dashboard or website. That way, your client has easy access to your documentation and has a quick way of contacting you in case they encounter a problem.

The plugin or it's author are not affiliated with Helpscout in any way.

== Installation ==

To use this plugin, you will need a [Helpscout](http://helpscout.net) account.

1. Upload `wp-helpscout-beacon` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under Tools/Helpscout Beacon, enter your Beacon Form ID and, if applicable, your Docs subdomain.
4. Check the approriate options ('enable contact form' and/or 'enable docs')
